# CartIQ 🛒

> **Give your online store a complete data-science team — in a single plug-in.**

CartIQ is an AI-powered plug-in designed for small and medium-sized online retailers. It layers **four Machine Learning modules** on top of existing storefronts, giving any e-commerce merchant access to capabilities previously reserved for large-scale tech retailers.

---

## 🧠 Four ML Modules

| Module | Algorithm | Type |
|--------|-----------|------|
| 🎯 Smart Product Recommendations | Gradient Boosting (XGBoost) + ANN | Supervised |
| 👥 Customer Segmentation | K-Means Clustering | Unsupervised |
| 📈 Demand Forecasting | Random Forest + Linear Regression | Supervised |
| 🛡️ Fraud & Fake Review Detection | Isolation Forest | Unsupervised |

## 🚀 Running the MVP Prototype Locally

### Prerequisites
- Python 3.9+
- pip

### Installation

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/cartiq-mvp.git
cd cartiq-mvp

# Install dependencies
pip install -r requirements.txt

# Run the Streamlit app
streamlit run cartiq_app.py
```

The app will open at `http://localhost:8501`
